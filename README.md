Visit our retro-style desktop interface:
```

```

### 2. Download Your Log
Extract your NGINX error log from your server:

**Linux/Ubuntu/macOS:**
```bash
scp user@yourserver:/var/log/nginx/error.log ./error.log
```

**Windows PowerShell:**
```powershell
scp user@yourserver:/var/log/nginx/error.log C:/Users/yourname/Downloads/error.log
```

### Supported Formats
- **Uncompressed Files**: `.log`, `.txt` files only

- **No Compression**: Please extract `.gz` files before upload
- **Text Format**: Plain text log files

```bash

/var/log/nginx/error.log          # Ubuntu/Debian
/var/log/nginx/error.log          # CentOS/RHEL
/usr/local/var/log/nginx/         # macOS Homebrew
/opt/nginx/logs/error.log         # Custom installations
```
